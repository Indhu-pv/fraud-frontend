# Fraud Detection System (Frontend)

A modern web dashboard for monitoring and analyzing fraudulent transactions.  
This project provides an intuitive interface to visualize fraud alerts, transaction data, and analytics.

---

## Project Overview

The Fraud Detection System Dashboard allows users to:

- Monitor suspicious transactions
- View fraud alerts
- Analyze transaction trends
- Track fraud statistics through charts

This project is designed as a frontend dashboard for a fraud detection platform.

---

## Tech Stack

Frontend technologies used:

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router
- Recharts (data visualization)

---

## Features

- Secure login page
- Fraud monitoring dashboard
- Transaction history view
- Fraud alert panel
- Interactive charts for analytics
- Responsive layout
